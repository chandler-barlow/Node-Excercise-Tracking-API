Excercises

GET all lifts

/api/excercises/all

Gets all excercises

GET all lifts in muscle group

/api/excercises/by_muscleGroup/:group

:group is the group you want to query

GET all lifts by name

/api/excercises/by_liftName/:name

:name is the name of the lift you want to query

POST log new excercise

/api/excercises/add

header:
content-type: application/json

body:
muscleGroup: name of the muscle group,
liftName: name of the lift to add
reps: the number of reps
weight: weight to add
userId: id of the user logging

PUT edit excercise log

/api/excercises/edit

header:
content-type: application/json

body:
id: id of the lift log to edit
muscleGroup: name of the muscle group,
liftName: name of the lift to add
reps: the number of reps
weight: weight to add

You don't have to include all of these fields in the body.
If a field isn't included in the call then that information will not change.

DELETE excercise log

/api/excercises/delete

header:
content-type: application/json

body:
id: id of excercise record to delete,
